# Clinical Style Reference

This file defines standard medical abbreviations, specialty terminology adaptations, and
ambiguity handling rules. Read this when the source uses heavy abbreviations or a specialty
mode is requested.

## Non-Standard Shorthand — Spell Out on First Use

If the doctor uses shorthand not on the list above, spell it out in parentheses on first
occurrence, then use the abbreviation thereafter:

> `Pt c/o L-sided HA (headache) for 3d (days)` →
> `Patient complaining of left-sided headache (HA) for 3 days`

If the abbreviation is ambiguous and context does not resolve it, note it:
`[abbreviation unclear — please verify]`

## Units and Values

- Always include units with clinical measurements: **140/90 mmHg**, **9.2 g/dL**, **37.2°C**
- Separate the number from the unit with a space: **82 bpm**, not **82bpm**
- Use the conventional local format for units where relevant (e.g., mg/dL vs mmol/L for glucose)
- If units are absent from the source: transcribe the number and add `[units not documented]`

---

## Drug Naming

- Prefer generic (INN) names unless the doctor used a brand name specifically
- If brand name used: `Augmentin (co-amoxiclav) 625 mg BD`
- If dose or frequency missing: `metformin — dose not documented`
- Never look up and substitute a "standard" dose — document only what the doctor stated

## Severity Grading Systems (use only if doctor provided the grading)

| System | Use case |
|---|---|
| CCS Class I–IV | Angina severity |
| NYHA Class I–IV | Heart failure functional capacity |
| Child-Pugh A/B/C | Liver disease severity |
| GCS 3–15 | Consciousness level |
| VAS / NRS 0–10 | Pain score |
| ECOG 0–5 | Performance status (oncology) |

Only include grading in the note if the doctor explicitly assigned it.

---

## Sensitive Information Handling

- Patient names and identifiers should be transcribed faithfully in the note — these are clinical records
- If the user asks for an anonymised version (e.g., for teaching), use `[Patient Name]`, `[DOB]`, `[ID]` as placeholders
- Never add patient identifiers that were not in the source
